# Koza Mobile App (React Native)

Bu proje Koza uygulamasının **React Native** (Expo) versiyonudur.

## Kurulum (ÖNEMLİ)

Bu projeyi çalıştırmak için bilgisayarınızda **Node.js** yüklü olmalıdır.

1.  **Node.js Yükleyin:** [https://nodejs.org/](https://nodejs.org/) adresinden indirin ve kurun.
2.  **Bağımlılıkları Yükleyin:**
    Bu klasörde (`koza-mobile`) terminal açın ve şunu çalıştırın:
    ```bash
    npm install
    ```

## Çalıştırma

Kurulum bittikten sonra uygulamayı başlatmak için:

```bash
npx expo start
```

*   Karşınıza çıkan QR kodu telefonunuzdaki **Expo Go** uygulaması ile okutarak test edebilirsiniz.
*   veya `a` tuşuna basarak (Android Emulator varsa) emülatörde açabilirsiniz.
